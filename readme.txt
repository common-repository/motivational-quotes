This plugin allows you to display random quotes on your posts and all registered users to see the list of all quotes.

== Description ==

Motivating Quotes allows you to display a randomly selected quote on any of your posts.
It comes with 10 quotes and you can add your own quotes from the option panel.
You can delete any quotes you don't like.
Every registered users can also search for all the quotes on your website.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/motivation-quotes` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Add to any of your post the shortcode [motivational-quote] to display a quotes.

== Frequently Asked Questions ==

= What is Motivating Quotes? =

It allows you to display your quotes on your posts.

= How to add my own quotes? =

In the administration panel you should have a line "Motivation quotes admin". Simply click on it and add a quote from the page that will be displayed.

= How to delete a quote? =

In the administration panel you should have a line "Motivation quotes admin". Simply click on it and write the quote you want to delete in the correct field.
